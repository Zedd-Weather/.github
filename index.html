Our Projects

Two complementary repositories that form a complete, open-source weather data platform.

🌩️
Zedd-Weather
Edge Telemetry & Risk-Analysis Dashboard

A full-stack edge platform running on a Raspberry Pi cluster. Combines real sensor hardware, on-device AI inference (Hailo-8L NPU), an MQTT telemetry pipeline, FastAPI backend, and a Dash/Plotly dashboard to deliver site-specific weather intelligence for industrial, agricultural, and construction operations.

Python Dash / Plotly FastAPI MQTT Hailo NPU Docker Raspberry Pi
⭐ 1 🔤 Python 📄 MIT
View repository →
🛰️
Zedd-Satellite
GIS Satellite Ground Station

A Raspberry Pi–based ground station that automatically tracks, captures, and decodes imagery from polar-orbiting weather satellites (NOAA 15/18/19 APT and Meteor-M2 LRPT) using an RTL-SDR dongle. Schedules passes via TLE data, records FM audio, and decodes it into satellite imagery — all autonomously.

Platform Features

Everything you need to monitor, analyse, and respond to on-site weather conditions.

🔬

Real Sensor Hardware

BCRobotics Weather HAT PRO, SparkFun anemometer, wind vane, and rain gauge — no synthetic or mock data, ever.

🧠

On-Device AI Inference

Hailo-8L NPU weather classifier plus local Ollama/Gemma for mitigation guidance — completely offline.

📡

MQTT Telemetry Pipeline

Sensor readings flow through Mosquitto (auth-required) to the control plane for real-time processing and storage in InfluxDB.

⚠️

Sector Risk Engines

Three purpose-built engines for construction, agricultural, and industrial operations with configurable alert thresholds.

🛰️

Satellite Imagery

Auto-schedule and decode passes from NOAA 15/18/19 and Meteor-M2 — from TLE orbital data to decoded PNG images.

📊

Dash / Plotly Dashboard

Rich interactive visualisations powered by Python Dash — no Node.js runtime, no npm build step.

🐳

Docker & Cluster-Ready

Single-host or three-node cluster deployment via Docker Compose with multi-arch (arm64 + amd64) images.

📻

LoRa Heartbeat Telemetry

SX1262 LoRa HAT broadcasts a compact JSON health snapshot every configurable interval — useful for off-grid monitoring.

🔒

Privacy & Security First

All third-party API keys stay server-side. The browser never sees credentials. CodeQL and CI scanning on every push.
